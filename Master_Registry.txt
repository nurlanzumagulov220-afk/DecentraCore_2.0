Ниже представлен структурированный реестр источников, очищенный от навигационного шума, элементов пользовательского интерфейса и повторяющихся фрагментов документации.

### 1. Документация библиотек и фреймворков (Offline-First)

*   **Dexie.js: Справочник API и основы** — Полное руководство по работе с оберткой для IndexedDB, включая работу с коллекциями, таблицами, транзакциями и схемами баз данных.
*   **Dexie Cloud: Быстрый старт и синхронизация** — Руководство по созданию приложений с поддержкой облачной синхронизации, управления доступом и аутентификации.
*   **Automerge (GitHub)** — Описание библиотеки на основе CRDT для создания JSON-подобных структур данных, которые могут редактироваться параллельно и объединяться автоматически.
*   **Collabs (Research Paper / arXiv)** — Научная статья о фреймворке для совместной работы, использующем механизмы «родительского контроля» для управления динамическими CRDT-объектами.
*   **@dexie-kit/sync** — SDK для синхронизации баз данных Dexie.js с любыми REST API, поддерживающее очередь оффлайн-запросов и разрешение конфликтов.
*   **Dexie Export/Import** — Аддон для Dexie.js, обеспечивающий экспорт и импорт данных базы в браузерной среде.

### 2. Законодательство и стандарты Республики Казахстан

*   **Закон РК «О персональных данных и их защите»** — Базовый закон, определяющий понятия персональных данных, биометрии, накопления, обезличивания и уничтожения информации.
*   **Правила сбора и обработки персональных данных (Приказ МЦРИАП № 395/НҚ)** — Регламент получения согласия субъекта, порядка хранения данных и работы сервисов контроля доступа.
*   **Единые требования в области ИКТ и информационной безопасности (Постановление № 119 от 25.02.2026)** — Обновленные стандарты РК, включающие требования к искусственному интеллекту, АСУ ТП и защите критически важных объектов.
*   **Национальные стандарты защиты информации (СТ РК 2025/2026)** — Перечень стандартов по криптографической защите (СТ РК 1073-2025) и критериям оценки безопасности ИТ-систем.
*   **Методика и правила испытаний объектов информатизации «Электронного правительства»** — Технический регламент проведения аудита кода, нагрузочных испытаний и проверки сетевой инфраструктуры.
*   **Перечень сроков хранения документов (Приказ Министра культуры и информации № 279-НҚ)** — Единый реестр, определяющий сроки хранения (от 1 года до «постоянно») для 788 видов документов.

### 3. Исследования и концепции Local-First

*   **Манифест Local-First Software (Ink & Switch)** — Статья 2019 года, определяющая семь идеалов разработки: отсутствие «спиннеров», работа без привязки к устройству, опциональность сети, безопасность и контроль пользователя.
*   **Local-First vs. Cloud-First (Update 2025)** — Сравнительный анализ архитектур, преимуществ владения данными и текущих технологических барьеров.
*   **Local-First приложения (блог ov7a)** — Разбор архитектурного пути данных (от DOM до диска) и проблем монетизации децентрализованного ПО.
*   **Подкаст Metamuse (Мартин Клеппман)** — Обсуждение CRDT, Automerge и будущего инструментов для совместной работы вне облака.
*   **Руководство для начинающих по Local-First (OpenReplay)** — Обзор современных браузерных API (OPFS, WebAssembly), делающих local-first архитектуру практичной.

### 4. Стандарты и технические протоколы

*   **RFC 9562 (UUIDs)** — Официальный стандарт IETF (май 2024), вводящий новые версии UUID (v6, v7, v8) с упорядочиванием по времени, идеально подходящие для распределенных баз данных.
*   **Consistency-Driven Data Synchronization Model** — Научная работа о моделях синхронизации данных для систем продажи билетов, работающих полностью оффлайн.

### 5. Корпоративные решения и платформы

*   **SAP Service and Asset Manager** — Обзор функций мобильного приложения для управления активами, включая механизмы автосинхронизации (CA_AUTO_SYNC) и работу с оффлайн-картами.
*   **Oracle REST Data Services (ORDS)** — Руководство разработчика по автоматическому созданию REST API для объектов базы данных Oracle и интеграции с JSON/OAuth.
*   **IBM Maximo Application Suite** — Материалы по стратегиям надежности, мониторингу IoT-устройств и интеграции с системами управления активами.
*   **Sage CRM SalesLogix Synchronization** — Описание технологии синхронизации данных для мобильных пользователей CRM через лог-файлы транзакций.

### 6. Экспертные обзоры и юридическая практика

*   **Юридический блог Femida Justice** — Анализ ответственности (административной и уголовной) за нарушение правил защиты персональных данных в Казахстане в 2025–2026 гг..
*   **Евразийский цифровой фонд (EDF)** — Информация о требовании локализации серверов с данными граждан Казахстана на территории РК с 2025 года.
*   **Портал Учёт.kz** — Разбор изменений в КоАП РК касательно увеличения штрафов за утечки данных (до 3000 МРП).
*   **Front-end Developer Handbook 2019** — Справочник по инструментам, браузерам и технологиям фронтенд-разработки.
